# markdown

　　之前用了很多Markdown 文档生成工具，发现有几个挺好用的，现在整理出来，方便大家快速学习。

* loppo: 非常简单的静态站点生成器
* idoc: 简单的文档生成工具
* gitbook: 大名鼎鼎的文档协作工具
* docsify: 一个神奇的文档站点生成器，简单轻巧，无需静态构建html
* hexo: 一个快速、简洁且高效的博客框架
